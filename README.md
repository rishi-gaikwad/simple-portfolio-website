
 <h3 align="center">Portfolio Clone</h3>

## Introduction:

This application is a Portfolio clone using React

## Start:

```sh
$ git clone https://github.com/rishi-gaikwad/Protfolio-Clone
```

```sh
$ cd Portfolio-clone
```

```sh
$ npm install
```

```
$ npm start
```
